# Country of Towers 🏰⚔️

A real-time card duel game built with React Native (Expo) and Colyseus. Two players battle to destroy each other's tower using a deck of 102 unique cards across three resource colors.

## 🎮 Game Overview

Each player starts with:
- **Tower:** 50 HP
- **Wall:** 10 HP
- **Resources:** 1 Mine (Ore), 1 Monastery (Mana), 1 Barracks (Troops)

On each turn, you:
1. **Collect** — Resources are generated by your buildings
2. **Play** — Play a card from your hand (if you can afford it)
3. **Draw** — Draw a card from your deck

### Win Conditions
- **Tower Demolition** — Opponent's tower reaches 0 HP
- **Tower Victory** — Your tower reaches 50+ HP
- **Resource Supremacy** — All 3 resources reach 150+

## 🃏 Cards

102 cards across 3 colors:

| Color | Resource | Count | Theme |
|-------|----------|-------|-------|
| 🔴 Red | Ore | 34 | Earth, walls, buildings, shields |
| 🔵 Blue | Mana | 31 | Arcane, tower boosts, monasteries |
| 🟢 Green | Troops | 37 | Creatures, damage, barracks |

Card effects include resource manipulation, direct damage, building construction, conditional effects, and special plays.

## 🛠️ Tech Stack

- **Client:** React Native (Expo) + TypeScript
- **Server:** Node.js + Colyseus + TypeScript
- **State Sync:** @colyseus/schema (delta-compressed binary)
- **Animations:** react-native-reanimated
- **Audio:** Built-in sound system
- **Testing:** Jest + Vitest

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repo
git clone https://github.com/pv-vimalnair/country-of-towers.git
cd country-of-towers

# Install client dependencies
npm install

# Install server dependencies
cd server
npm install
cd ..
```

### Running Locally

1. **Start the server:**
```bash
cd server
npm run dev
```

2. **Start the client:**
```bash
# In a new terminal
npx expo start
```

3. Open the app on your device or emulator. Play against yourself using two browser tabs.

## 🧪 Testing

```bash
# Run card engine tests
cd server
npm test
```

## 📊 Matchmaking

ELO-based matchmaking with a 30-second turn timer. Reconnection supported — if you disconnect, you have 60 seconds to rejoin.
