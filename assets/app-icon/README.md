# App Icon — Design Brief for Rachana

## Overview
Castle/tower silhouette on dark purple background. Should feel epic, cosmic, medieval.

## Dimensions
Source: **1024×1024px** PNG (transparent background for iOS, solid dark purple for Android adaptive)

## Color Spec
- Background: `#0F172A` (dark slate) or `#1E1B4B` (deep indigo)
- Tower accent: `#F59E0B` (gold) or `#A855F7` (purple glow)
- Style: Flat silhouette with subtle gradient glow behind tower
